# Python Functions and OOPs Assignment

This repository contains Python code examples that demonstrate the concepts of **Functions** and **Object-Oriented Programming (OOP)** in a simple and practical way.

The assignment includes:
- Function definitions and usage
- Lambda and generator functions
- Basic and advanced OOP concepts
- Encapsulation, Inheritance, Polymorphism, and Abstraction
- Use of decorators, iterators, and class/static methods

Each concept is explained with straightforward code to help with understanding and learning.

## How to Use

You can open the code files or notebooks in your preferred Python environment (like Jupyter Notebook or Google Colab) and run the examples to see how they work.
